# Super Bug Zapper Game!

A raw WebGL game where you rotate a 3D sphere covered in bacteria and click them before they grow too big.

## Play it

[https://harkirat-c.github.io/SuperBugZapper/]

## Files

- `index.html` — page layout and styling
- `a4.js` — game logic, WebGL rendering, and input handling
- `lib/` — small WebGL helper libraries (matrix math, shader utils, etc.)

Made by Harkirat Chahal.
